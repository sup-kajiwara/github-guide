# イラストの出典・ライセンス

このフォルダのイラストは **unDraw**（作者: Katerina Limpitsouni）のものです。

- 公式サイト: <https://undraw.co/>
- ライセンス: **MITライク（オープンライセンス）** … 商用利用OK・**帰属（クレジット）表記は不要**
- 取得元（SVGコレクション・MIT）: <https://github.com/balazser/undraw-svg-collection>

!!! note "色について"
    元のSVGは色をCSS変数 `--primary-svg-color` で指定していますが、
    画像として確実に表示されるよう、基調色を `#6c63ff`（unDrawの標準色）に固定してあります。
    テーマに合わせて色を変えたい場合は、各SVG内の `#6c63ff` を一括置換してください。
