# 画面キャプチャ（スクリーンショット）置き場

このフォルダに、各ページで使う画面キャプチャ画像を入れます。

## 使い方

1. 画面を撮影して、わかりやすい名前で保存します
   - 例：`account-01-signup.png`（章名-連番-内容）
2. 本文の「📷 画面キャプチャ枠」と書かれた場所に、次のように差し込みます

   ```markdown
   ![Sign upボタンの場所](images/account-01-signup.png){ width="700" }
   ```

   - `{ width="700" }` で横幅を指定できます（省略可）

!!! tip "撮影のコツ"
    - 押すボタンの周りを **赤枠** で囲むと、どこを操作するか一目で伝わります
    - 個人情報（実在のメールアドレス・氏名）が写り込まないよう注意してください
