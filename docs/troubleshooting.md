## 用語・ボタン早見表

| 英語表記 | 意味 |
|---|---|
| Repository | リポジトリ（ファイル置き場） |
| Commit | 変更を記録する |
| Push | アップロードする |
| Pull | 取り込む |
| Branch | 作業用の枝分かれ |
| Merge | 本体に合流する |
| Pull Request | 変更の確認依頼 |

!!! tip "それでも解決しないとき"
    社内の詳しい人に聞く、または公式ヘルプ（日本語あり）を参照しましょう。
